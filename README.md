# PulseMate - AI Healthcare Monitoring & Incident Management System

## 🚀 Overview
PulseMate is an AI-powered healthcare monitoring and incident automation platform integrating ServiceNow, LangGraph, and RAG.

## 🧠 Features
- Role-based authentication (User/Admin)
- Patient vitals monitoring
- AI chatbot (RAG-based)
- ServiceNow ticket automation
- LangGraph workflow automation
- Offline fallback support

## 🏗 Tech Stack
Frontend:
- React 18 + Vite + Tailwind + TypeScript

Backend:
- FastAPI (LangGraph agent)
- Node.js (Express)

AI:
- Azure OpenAI
- ChromaDB (RAG)

Database:
- SQLite

## 🤖 Chatbot Capabilities
- App navigation help
- Troubleshooting
- Ticket creation
- Auto-resolution using AI

## 🔄 Architecture
- Microservices-based design
- Agentic workflow automation
- Role-based access control

## 🧪 Testing
Includes 30+ test cases for:
- Authentication
- Chatbot
- Ticket lifecycle
